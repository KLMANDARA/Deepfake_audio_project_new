# Deepfake Audio Detection (Production-Level)

## Features
- Waveform, Spectrogram, MFCC Visualization
- CNN on Spectrogram Images
- Training + Evaluation (Accuracy, Confusion Matrix)
- Prediction script
- Ready for GitHub + Resume

## Run
pip install -r requirements.txt

# visualize audio
python visualize.py

# train model
python train.py

# evaluate
python evaluate.py

# predict
python predict.py
